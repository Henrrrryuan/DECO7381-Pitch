Problem Space & Ethical Considerations

Cognitive Accessibility Assistant for Web Content

Web accessibility often focuses on sensory or motor impairments, while cognitive accessibility for neurodivergent users remains under-addressed. This project aims to identify cognitive load sources and provide clear, actionable recommendations.

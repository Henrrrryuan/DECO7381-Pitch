Problem Space & Ethical Considerations

Problem Space

Web accessibility focuses on visual/auditory/motor impairments, while cognitive accessibility for neurodivergent users is severely under-addressed.

  • Dense language, cluttered layouts, distracting animations increase cognitive load
  • Existing tools only check technical compliance (alt text, color contrast) – no cognitive analysis
  • WCAG/ISO 9241-11 provide principles but lack practical, actionable tools

Ethical Considerations

Core ethical safeguards integrated into the project design from the outset:

  • Data privacy: No user data collection, only static web content analysis
  • Equity: Prioritise neurodivergent user needs in all design decisions
  • Transparency: Clear explanation of cognitive load issues and optimization recommendations
  • Accessibility: The tool itself follows cognitive accessibility design principles

⚡ Interaction Complexity

  • Built-in Validation: Pre/post-optimization comparison + cognitive load measurement
